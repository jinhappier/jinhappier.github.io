기본 HTML태그

2020-04-06

기본 HTML 태그들

  • <!DOCTYPE>: html 버전 명시

  • <title>: 브라우저의 탭이나 방문 기록에 나와 있는 제목s

  • <h1>~<h6>: 머리말(heading)

  • <p>: 문단(paragraph)

굵은 텍스트

  • <b>: 시각적으로 텍스트를 굵게 만들 목적 (bold)
  • <strong>: 감싸고 있는 텍스트가 중요하다고 표시 (스크린리더가 강조해서 읽음)

날려쓰기

  • <i>: 시각적으로 날려쓰는 것이 목적(italics)
  • <em>: 강조하는 것이 목적 (emphasized)

하이퍼링크 태그

  • <a href="가고싶은 주소">내용</a>: Anchor
    • <a href="https://google.com">구글로가는 링크</a>:

기본 CSS 속성들

  • <style>: html 안에서 style태그로 css스타일을 적용할 수 있다

  • 폰트크기: <px>
