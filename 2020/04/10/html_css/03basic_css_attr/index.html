기본 CSS 속성들

  • <style>: html 안에서 style태그로 css스타일을 적용할 수 있다

  • 모든 사이즈 설정: px, %

    • 절대적인 폰트 크기(Absolute): px, pt
    • 상대적인 폰트 크기(Relative): em, %

  • 텍스트 색 color [W3 CSS colors]

    • color: blue;: CSS에서 정해준 색 이름 중 하나를 골라쓰는 방법
    • color: rgb(83, 237, 65);: 빨강(Red), 초록(Green), 파랑(Blue)를 수치화하여 섞인 색 표현방식
    • color: #53ED41;: HEX값(16진법), 83은 16진법으로 53, 237은 ED, 65는 41, 즉 rgb(83, 237, 65)와 같음

  • 텍스트 굵기 font-weight

    • font-weight: 100/200/300/400/500/600/700/800/900; 만 가능하지만 태그마다 지원해주는 텍스트 굵기가 있기 때문에 적용이 안될 수도 있다.

  • 텍스트 정렬 text-align

    • text-align: right/left/center;

  • 텍스트 줄 삽입 or 제거 text-decoration

    • text-decoration: underline/overline/line-through/none;

  • 줄간격 조절 line-height

    • line-height: normal/90px;

폰트 파일 사용하기

  • 맘에 드는 폰트를 다운로드 후 프로젝트 폴더에 fonts라는 폴더를 생성 후 해당 폴더에 다운로드 받은 파일을 넣는다.

  • @font-face {
  src: url("폰트가 있는 경로");
  font-family: "Jua"; /* 원하는 폰트 이름 */
}

p {
  font-family: "Jua";
}

inline VS block

  • inline속성인 태그일 경우 text-align 속성이 먹지 않는다. 그 이유는 해당 태그가 차지하는 공간이 글자가 차지하는 공간뿐이기 때문이다. 이럴 경우 텍스트의 위치를 조정하고 싶을 때 해당 태그를 block속성인 <div>로 묶어주면 text-align속성이 적용된다. (c.f. <span>은 inline속성)
