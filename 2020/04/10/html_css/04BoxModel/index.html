Box Model

2020-04-10

padding

: 내용과 테두리 사이 여유 공간(테두리 안의 공간)

border

: 테두리

border-radius

: 요소의 모서리를 둥글게 만듦

margin

: 이 요소와 다른 요소 사이의 여백 (테두리 밖의 공간)

width, height, min, max

: 요소의 가로와 세로의 길이 설정

overflow

: width, height, max-width, max-height등을 설정하다보면, 내용물이 들어갈 공간이 부족한 경우에 해당 속성으로 처리

background-color

: 배경색 설정 (색이름, RGB코드, HEX코드 중 하나 입력)

background-repeat

: 배경 이미지를 반복시킬 것인지 아닐 것인지, 그리고 반복시킨다면 어떤 방식으로 반복시킬 것인지 정해주는 속성

background-size

: 배경이미지의 사이즈를 정해주는 속성

background-position

: 배경 이미지의 위치를 정해주는 속성

box-shadow

: 요소에 그림자 설정

box-sizing

위의 두 박스는 넓이와 높이가 같지만 테두리와 패딩을 고려해서 계산해주어야 하는 불편함이 있다.

CSS3부터는 box-sizing속성을 사용하면 이 문제를 해결할 수 있다.

box-sizing의 기본값은 content-box인데, 이걸 border-box로 바꿔보자
